# Générateur de mots de passe – Web App

## Fonctionnalités

- Choix de la **longueur** du mot de passe (4 à 128 caractères)
- Sélection des types de caractères à inclure :
  - Uppercase (A, B, C, …)
  - Lowercase (a, b, c, …)
  - Chiffres (0, 1, 2, …)
  - Minus (-)
  - Underline (_)
  - Space ( )
  - Caractères spéciaux (!, $, %, …)
  - Brackets ([, ], {, }, (, ))
- Champ pour **caractères personnalisés**
- Bouton **Copier** le mot de passe dans le presse-papier

## Utilisation

1. Ouvrez le fichier `index.html` dans votre navigateur **ou hébergez sur GitHub Pages**.
2. Choisissez la **longueur** et les **types de caractères** à inclure.
3. Ajoutez éventuellement des **caractères personnalisés**.
4. Cliquez sur **Générer mot de passe**.
5. Cliquez sur **Copier** pour l’envoyer dans le presse-papier.
